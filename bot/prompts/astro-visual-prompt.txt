ФАЗА 2: ГЕНЕРАЦИЯ ФИНАЛЬНОГО ПРОМТА
═══════════════════════════════════════════════════════════════

Создай ОДИН промт для генерации изображения на АНГЛИЙСКОМ языке.

Промт должен включать (без астрологических терминов!):
1. Описание стиля/медиума (определён знаком Солнца)
2. Цветовую палитру (5-7 конкретных цветов из карты)
3. Композицию (определена домом Солнца)
4. Освещение (определено Луной)
5. Атмосферу/настроение (из аспектов)
6. Символические элементы (из Атмакараки и накшатры)
7. Если есть фото лица — инструкция сохранить лицо

ФОРМАТ ВЫВОДА:
Верни ТОЛЬКО промт для генерации изображения. Ничего больше. Без объяснений, без кавычек.
Промт на АНГЛИЙСКОМ. 200-400 слов.
