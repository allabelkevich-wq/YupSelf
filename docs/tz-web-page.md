# ТЗ: Веб-страница YuPself — AI-генерация изображений

## Суть проекта

YuPself — сервис генерации AI-изображений. Пользователь описывает картинку текстом или голосом, AI улучшает промт и генерирует изображение на основе модели NanoBanana Pro (Gemini 3 Pro).

Нужна одностраничная веб-версия сервиса — красивая, современная, на русском языке. Это не лендинг, а рабочий интерфейс для генерации.

---

## Структура страницы

### 1. Шапка
- Логотип / название **YuPself**
- Подзаголовок: «Создавай изображения силой слова»
- Минималистичная, без лишних меню

### 2. Главный блок — генерация
Центр страницы. Это самое важное.

**Поле ввода:**
- Большое текстовое поле с плейсхолдером: «Опиши, что хочешь увидеть...»
- Примеры внутри или под полем:
  - «Кот-астронавт на поверхности Луны»
  - «Закат над океаном в стиле масляной живописи»
  - «Логотип кофейни в стиле минимализма»
- Кнопка микрофона справа от поля (голосовой ввод)

**Настройки генерации (компактная панель под полем):**

Три ряда кнопок-чипов (pill-shaped, выбор одного в каждом ряду):

**Стиль:**
- Фотореализм
- Эдиториал
- Кинематограф
- Аниме
- Масло
- Концепт-арт
- Акварель
- 3D рендер
- Минимализм
- Сюрреализм

**Формат:**
- 1:1 Квадрат
- 9:16 Stories
- 16:9 Широкий
- 4:5 Instagram
- 3:4 Портрет
- 2:3 Постер
- 21:9 Кино

**Разрешение:**
- 512px (быстрое)
- 1K (стандарт)
- 2K (высокое)
- 4K (максимум)

**Кнопка генерации:**
- Крупная, заметная: «Сгенерировать»
- При нажатии — состояние загрузки (спиннер или анимация)

### 3. Блок результата
- Появляется после генерации
- Сгенерированное изображение в крупном виде
- Под изображением:
  - Кнопка «Скачать» (скачать в полном разрешении)
  - Кнопка «Повторить» (та же идея, новый результат)
  - Кнопка «Новое описание» (очистить и начать заново)
- Показать использованный промт (свёрнутый аккордеон «Промт», чтобы видеть что AI сгенерировал)

### 4. Галерея примеров (опционально)
- 6–8 карточек с примерами генераций
- При наведении — показать промт
- Клик — вставляет промт в поле ввода

### 5. Футер
- «YuPself — AI Image Generation»
- Ссылка на Telegram-бота: @YupSelf_bot
- Год: 2026

## Дизайн

### Общий стиль
- **Тёмная тема** — основной фон: глубокий тёмно-синий / почти чёрный (#0a0a1a)
- **Акцентный цвет**: градиент от фиолетового к синему (#7c3aed → #3b82f6)
- **Типографика**: современный sans-serif (Inter, Manrope или аналог)
- **Стеклоэффекты**: glassmorphism на карточках и панелях (backdrop-blur, полупрозрачный фон)
- **Анимации**: плавные, сдержанные. Кнопки — hover с мягким свечением. Результат — fade-in
- **Нет агрессивных градиентов, неона, радуги** — премиальный, спокойный вид
- **Мобильная адаптация**: страница должна хорошо выглядеть на телефоне (основной трафик)

### Кнопки-чипы (стили, форматы, разрешения)
- Pill-shaped (border-radius: 9999px)
- Неактивная: полупрозрачный фон, тонкая рамка
- Активная: заливка акцентным градиентом, без рамки
- Все одинакового размера в своём ряду

### Кнопка «Сгенерировать»
- Полная ширина поля ввода
- Градиентный фон (акцентные цвета)
- Крупный текст, border-radius: 9999px
- При загрузке: текст меняется на «Генерирую...» + спиннер

### Поле ввода
- Тёмный фон, светлый текст
- Высота: минимум 3 строки, растягивается по контенту
- Мягкая подсветка при фокусе (glow акцентным цветом)

## Технические требования

- Одностраничное приложение (SPA)
- HTML + CSS + JavaScript (без фреймворков, чистый код)
- API-запросы на бэкенд: `POST https://yupself-bot.onrender.com/api/generate`
  - Body: `{ "prompt": "текст", "style": "...", "aspectRatio": "1:1", "imageSize": "1K" }`
  - Response: `{ "imageBase64": "...", "enhancedPrompt": "..." }`
- Голосовой ввод через Web Speech API (браузерный) или отправка аудио на сервер
- Адаптивная вёрстка (mobile-first)
- Язык интерфейса: русский
